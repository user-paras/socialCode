This project is a TypeScript/React application built with Vite, utilizing Tailwind CSS for styling and Supabase for backend services.

## Prerequisites
Before you begin, ensure you have the following installed:
1.  Node.js and npm (or yarn/pnpm)
2.  Git

## Setup Instructions
    Navigate to the project root directory and install the required Node.js packages.

    ```bash
    npm install
    # or yarn install, or pnpm install depending on your package manager
    ```

## Running the Application
To start the development server:

```bash
npm run dev
# or yarn dev, or pnpm dev
```

This command typically starts the Vite development server, usually accessible at `http://localhost:5173` or a similar local address.
